# Traducción de Mensajes de Error a Castellano

## RESUMEN

Este documento detalla todos los mensajes de error que se van a traducir al castellano en la API.

## TRADUCCIONES APLICABLES

### Mensajes Comunes

-   `'error' => 'Error al obtener datos'` → `'error' => 'Error al obtener datos'` ✅ (ya en español)
-   `'error' => 'Usuario no encontrado'` → ✅ (ya en español)
-   `'error' => 'Acceso denegado'` → ✅ (ya en español)
-   `'error' => 'Fecha inválida'` → ✅ (ya en español)

### Mensajes que NECESITAN Traducción

#### Inglés → Español

1. `'No encontrado'` → OK
2. `'Perfil no encontrado'` → OK
3. `'No tienes permiso para editar este perfil'` → OK
4. `'El usuario (correo/DNI) ya existe'` → OK
5. `'Error registro'` → `'Error de registro'`
6. `'Error al calcular recomendaciones'` → OK
7. `'No estás inscrito en esta actividad'` → OK
8. `'No se pudo realizar la inscripción'` → OK
9. `'No se pudo realizar la inscripción por reglas de negocio'` → OK

#### Fix de mensajes confusos/en inglés

-   `'Usuario Organización no encontrado'` → `'Organización no encontrada'`
-   `'The cupo de la actividad está lleno (Overbooking prevented)'` → `'El cupo de la actividad está lleno'`
-   `'Error al cargar catálogo'` → OK
-   `'Falta la URL de la imagen'` → OK
-   `'Error al guardar imagen'` → `'Error al guardar la imagen'`
-   `'Faltan datos (id_idioma, nivel)'` → OK
-   `'El voluntario ya tiene asignado este idioma. Usa PUT para modificar el nivel.'` → OK
-   `'Este voluntario no tiene asignado ese idioma'` → OK
-   `'Recurso no encontrado'` → OK
-   `'Rol de usuario inválido para esta acción.'` → `'Rol de usuario inválido para esta acción'`
-   `'Estado de cuenta inválido. Use: ...'` → `'Estado de cuenta inválido. Usa: ...'`
-   `'Estado de publicación inválido. Use: ...'` → `'Estado de publicación inválido. Usa: ...'`
-   `'Error al eliminar actividad'` → `'Error al eliminar la actividad'`
-   `'Actividad editada por Coordinación'` → OK
-   `'Error al cerrar cuenta'` → `'Error al cerrar la cuenta'`
