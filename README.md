# Proyecto Voluntariado - API REST

Este documento sirve como guía general sobre la estructura de carpetas de este repositorio.

## Estructura del Repositorio

- **SQL_Querys_VoluntariadoDB_Backend/**:
  - Esta carpeta contiene la estructura inicial y los scripts SQL utilizados para diseñar y crear la arquitectura de la base de datos y la API.

- **SYMFONY_API_Voluntariado4v/**:
  - Esta carpeta contiene el proyecto de la API desarrollado en Symfony.
  - **Importante**: Dentro de esta carpeta (específicamente en `api_voluntariado_4v`) encontrarás el **README oficial del proyecto**, el cual incluye toda la documentación técnica, instrucciones de instalación, configuración y uso de la API.

## Cómo empezar

Para poner en marcha el proyecto, por favor navega a la carpeta `SYMFONY_API_Voluntariado4v/api_voluntariado_4v` y sigue las instrucciones detalladas en su archivo `README.md`.
